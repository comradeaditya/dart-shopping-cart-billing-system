# Shopping Cart & Billing System - Dart

## About
A console-based Shopping Cart & Billing System 
built using Dart fundamentals.

# Features
-  Display available products
-  Add items to cart
-  Stock management
-  Itemized bill calculation
-  Discount & GST calculation
-  Final invoice generation
-  Exception handling for wrong inputs

##  Dart Concepts Used
- List & Map (Collections)
- Functions & Arrow Functions
- For/ForEach Loops
- Conditions (if/else)
- Exception Handling (try/catch)
- DateTime for invoice timestamp
- firstWhere() for product search
- String Interpolation
